### **Instalasi & Menjalankan 🚀**

1.  Clone repositori:

    ```bash
    git clone https://github.com/FrostLeaf15/BelajarReact
    ```
2.  Masuk ke direktori:

    ```bash
    cd BelajarReact
    ```
3.  Install dependensi:

    ```bash
    npm install
    ```
4.  Jalankan proyek:

    ```bash
    npm run dev
    ```
